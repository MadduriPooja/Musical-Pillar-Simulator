# Musical Pillar Simulator 🎶

## Inspiration
This program is inspired by the *Musical Pillars of Vittala Temple, Hampi*.  
The temple is unique because its stone pillars produce resonant musical sounds when tapped.  
I wanted to connect this speciality with coding and explore how data structures can represent such ideas.

## About the Program
- Written in **Java**.  
- Each pillar ID (P1, P2, …) is mapped to a musical note (Sa, Re, Ga, Ma) using **HashMaps**.  
- When the user enters a sequence of pillar IDs, the program outputs the corresponding notes.  
- It also checks if the sequence matches a predefined **raga** using arrays and string matching.

## DSA Concepts Used
- **HashMap** → mapping pillar IDs to notes.  
- **Arrays/Lists** → storing sequences of notes.  
- **Pattern Matching** → detecting ragas.
